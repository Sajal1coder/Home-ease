## ğŸš€ **Getting Started**

### **Prerequisites**
- Node.js 18+ and npm
- MongoDB (local or Atlas)
- Docker Desktop (for chatbot)
- Stripe Account
- Google Gemini API Key (for chatbot)

### **1. Clone the Repository**

```bash
git clone https://github.com/Sajal1coder/Home-ease.git
cd Home-ease
```

### **2. Backend Setup**

```bash
cd server
npm install
```

Create `.env` file in `server/` directory:

```env
# Database
MONGO_URI=mongodb+srv://your-mongodb-uri
MONGO_URL=mongodb+srv://your-mongodb-uri

# Authentication
JWT_SECRET=your-jwt-secret-key

# Stripe Payment
STRIPE_PUBLISHABLE_KEY=pk_test_your-publishable-key
STRIPE_SECRET_KEY=sk_test_your-secret-key

# Server
PORT=3001
```

Start the server:
```bash
npm start
```

### **3. Frontend Setup**

```bash
cd client
npm install
```

Create `.env` file in `client/` directory:

```env
REACT_APP_API_URL=http://localhost:3001
REACT_APP_STRIPE_PUBLISHABLE_KEY=pk_test_your-publishable-key
```

Start the React app:
```bash
npm start
```

The application will open at `http://localhost:3000`

### **4. Chatbot Setup (Optional)**

The AI chatbot requires additional services. Follow the detailed guide:

```bash
cd chatbot
```

See complete setup instructions in:
- [`chatbot/README.md`](./chatbot/README.md) - Technical documentation
- [`chatbot/SETUP-GUIDE.md`](./chatbot/SETUP-GUIDE.md) - Step-by-step guide
- [`CHATBOT-OVERVIEW.md`](./CHATBOT-OVERVIEW.md) - Feature overview

**Quick Start:**
```bash
# 1. Configure environment
cp .env.example .env
# Edit .env with your Gemini API key

# 2. Start Docker services
docker-compose up -d

# 3. Load chatbot data
cd data-ingestion
npm install
npm run ingest

# 4. Access n8n workflow editor
# Open http://localhost:5678
# Import workflow from workflows/homeease-rag-chatbot.json
```

### **5. Database Seeding (Optional)**

Populate the database with sample data:

```bash
cd server
npm run seed        # Seed all data
npm run seed:users  # Seed users only
npm run seed:properties  # Seed properties only
```

### **6. Create Admin User**

Run in MongoDB shell or Compass:
```javascript
db.users.updateOne(
  { email: "your-email@example.com" },
  { $set: { isAdmin: true, verified: true } }
)
```

## ğŸ› **Known Issues & Troubleshooting**

### **Common Issues**

**CORS Errors:**
- Ensure backend CORS is configured with frontend URL
- Check `server/index.js` CORS settings

**Stripe Payment Fails:**
- Verify Stripe keys are correct (test vs live mode)
- Check webhook endpoint configuration

**Chatbot Not Responding:**
- Ensure Docker services are running: `docker-compose ps`
- Check n8n workflow is activated
- Verify Gemini API key is valid

**Socket.io Connection Failed:**
- Check firewall settings
- Verify Socket.io client URL matches server URL
- Ensure server is running on correct port

**Image Upload Fails:**
- Check `server/public/uploads` directory exists
- Verify Multer configuration
- Check file size limits
